How to install chrome

Welcome to the instructions on how to install google chrome, this is the softaware that should be installed before anything elese.

The instructions on here may be different and have diffrent steps due to the fact that we will be working with the ubuntu download and the windows 11 download

Windows 11 instuctions on how to download google chrome

  1. Open internet explorer
  2. Type “google chrome download”
  3. Go to the website and download the chrome download
  4. Once finish open it and follow what it says
  5. Follow what ever the wizard says and make sure you agree to all

Ubuntu imstructions on how to download google chrome

  1. Open terminal
  2. Type “sudo apt update”
  3. Type “sudo apt install google-chrome-stable”
  4. After you will be prompt to confirm installation, confirm the installation when given
